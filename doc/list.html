Подключение к проекту

Дли использования данного API в своём проекте необходимо просто подключить заголовочный файл list.h

#include "list.h"

Структуры данных

Списки определяются в виде структур типа list. Описания структуры приведены в заголовочном файле list.h. Прямое обращение к элементам этой структуры крайне нежелательно и результат такого обращения не гарантирован. Для получения данных необходимо использовать соответствующий API.

Также используется структура atomic . Эту структуру возвращает, например, функция car(struct list*). Структура имет вид

tupedef struct {
void* data;
enum typelem type;
} atomic

где type принимает значения ATOM для атомарных значений, LIST - для списков и NONE для отсутствующих значений (в данном контексте не используется)

API

car

atomic car(struct list*)
Принимает указатель на список и возвращает значение первого элемента

cdr

struct list* cdr(struct list*)
Принимает указатель на список и возвращает указатель "остаток" списка (список без первого элемента)

create

struct list* create(void)
Создает пустой список

get

get(atom)
Макрос. Принимает элемент atom типа atomic и возвращает указатель на его значение.

lfree

void lfree(struct list*)
Принимает указатель на список и освобождает его (список целиком)

push

push(elem, type, typelem, list)
Макрос. Принимает значение elem типа type с указанием целевого типа typelem(LIST или ATOM) и добавляет его в список list

pop

pop(list)
Макрос. Принимает указатель на указатель(!) на список и возвращает указатель на первый элемент, попутно сдвигая указатель исходного списка на следующий элемент
